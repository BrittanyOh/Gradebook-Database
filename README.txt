# README.txt

To run the gradebook database
1. Open up your mySQL shell
2. type '\sql' to enter SQL mode
3. Copy the contents from 'gradebook.sql' and paste in shell
4. Copy the contents from 'init.sql' and paste in shell to set initial test values for the tables
5. Copy the contents from 'main.sql' to run the commands needed to answer questions 4-12
